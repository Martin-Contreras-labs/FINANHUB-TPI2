Ver la aplicacion en GOOGLE IA STUDIO: https://ai.studio/apps/drive/1AyICEHBBPw3SpYaPVX2Ve8EgZeLLHuW8

## Correr localmente

**Prerequisito:**  Node.js


1. Instalar paquete:
   `npm install`
2. Haz tu `GEMINI_API_KEY` en [.env.local](.env.local) en tu Gemini App Key
3. Correr la app:
   `npm run dev`
